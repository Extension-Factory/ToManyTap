# 📑 ToManyTab 사용 설명서

> "탭의 홍수에서 탈출하세요" - 모든 탭을 한 번에 저장하고 관리하세요!

---

## 🚀 시작하기

### 1단계: 확장 프로그램 로드
1. Chrome 열기
2. 주소창에 `chrome://extensions/` 입력
3. 우상단 **"개발자 모드"** 토글 켜기
4. **"압축 해제된 확장 프로그램을 로드합니다"** 클릭
5. `C:\WorkSpace\Project\12Month12Project\1month\ToManyTab\dist` 폴더 선택
6. ✅ **ToManyTab** 확장 프로그램이 설치됨!

### 2단계: 아이콘 찾기
- Chrome 우상단에 **📑 아이콘**이 보이면 성공!
- 이 아이콘을 클릭하면 ToManyTab 팝업창이 열립니다

---

## 📖 기본 사용법

### 🎯 탭 저장하기 (Stash)

#### 단계별 가이드:

1. **여러 개의 탭 열기**
   - 웹사이트들을 여러 개 탭으로 열어놓기
   - 예: 쇼핑, 뉴스, YouTube 등 자신이 원하는 탭들

2. **ToManyTab 아이콘 클릭**
   - Chrome 우상단의 📑 아이콘 클릭
   - 팝업창이 열림

3. **현재 탭 수 확인**
   - 팝업창 상단에 "📊 12 Tabs Open" 같은 메시지가 표시됨
   - 이것이 저장될 탭의 개수

4. **"⬇ Stash All & Close" 버튼 클릭**
   - 큰 파란 버튼을 클릭
   - 모든 탭이 **한 번에 닫히고 저장됨**

5. **세션 이름 입력 (선택사항)**
   - 팝업창이 입력창으로 변함
   - 예: "쇼핑", "프로젝트 A", "오늘 할 일" 등
   - 입력하지 않으면 자동으로 날짜/시간이 이름이 됨
   - ✓ 버튼으로 확인, ✕ 버튼으로 취소

6. **완료! 🎉**
   - "🎉 12 tabs stashed!" 메시지가 나타남
   - 팝업창의 목록에 새로운 세션이 추가됨

**결과:**
- ✅ 모든 탭이 닫힘 (PC 메모리 절약!)
- ✅ 탭 정보가 저장됨 (나중에 복구 가능)
- ✅ 브라우저가 깨끗해짐

---

### 🔄 저장된 탭 복구하기 (Restore)

#### 단계별 가이드:

1. **ToManyTab 아이콘 클릭**
   - 팝업창 열기

2. **저장된 세션 목록 보기**
   - 팝업창의 아래쪽에 저장된 세션들이 카드 형태로 표시됨
   - 각 카드 정보:
     - 📄 세션 이름 (위에 크게 표시)
     - 📅 저장한 날짜/시간
     - 📌 몇 개의 탭이 저장되었는지 (오른쪽 하단의 숫자 배지)
     - 🎨 저장된 웹사이트 아이콘들 (왼쪽부터 최대 5개)

3. **복구하고 싶은 세션 찾기**
   - 여러 세션이 있으면 상단부터 최신순으로 표시됨
   - 원하는 세션의 **"↻ Restore" 버튼** 클릭

4. **탭이 복구됨! ✅**
   - "✅ 12 tabs restored!" 메시지 표시
   - Chrome의 백그라운드에서 모든 탭이 자동으로 열림
   - (주의: 현재 창에 탭이 쌓여서 열리므로 Chrome 창의 탭 바를 확인하세요)

---

### 👁️ 세션 미리보기 (Preview)

저장된 탭들이 뭔지 확인하고 싶을 때 사용합니다.

#### 단계별 가이드:

1. **세션 카드의 "👁️ Preview" 버튼 클릭**
   - 세션 카드의 세 번째 버튼 클릭
   - 카드가 펼쳐져서 저장된 탭 목록이 나타남

2. **탭 목록 확인**
   - 각 탭의:
     - 🎨 웹사이트 아이콘
     - 📄 웹사이트 제목
     - 📍 URL 주소

3. **탭 표제 클릭**
   - 목록에서 특정 탭 제목을 클릭하면 그 탭만 바로 열림
   - (Restore 버튼 없이 개별적으로 열고 싶을 때 유용)

4. **닫기**
   - "👁️ Preview" 버튼을 다시 클릭하면 목록이 닫힘

---

### 🗑️ 세션 삭제하기 (Delete)

더 이상 필요없는 세션을 제거합니다.

#### 단계별 가이드:

1. **삭제하고 싶은 세션 찾기**

2. **세션 카드의 "🗑️" 버튼 클릭**
   - 붉은 색 휴지통 아이콘

3. **확인!**
   - "🗑️ Session deleted" 메시지 표시
   - 세션이 목록에서 사라짐
   - (주의: 삭제 후에는 복구할 수 없습니다!)

## 💡 사용 팁

### 팁 1️⃣: 상황별로 세션 나누기
```
예시:
- "쇼핑" - 물품 검색 탭들
- "프로젝트" - 업무 관련 탭들
- "공부" - 학습 자료 탭들
```

세션 이름 자동 입력으로, 같은 이름의 세션들을 쉽게 관리할 수 있습니다.

### 팁 2️⃣: 메모리 절약
- 안 쓰는 탭 세션을 자주 저장하세요
- RAM 사용량이 크게 줄어들어 PC가 빨라짐
- 특히 저사양 노트북에서 효과적!

### 팁 3️⃣: Quick Restore
- Preview 버튼으로 먼저 어떤 탭인지 확인하고
- Restore 버튼으로 전체 복구하기

### 팁 4️⃣: 정기적 정리
- 필요없는 세션은 정기적으로 삭제
- 저장소 공간 확보

### 팁 5️⃣: 여러 세션 활용
예를 들어:
1. "아침 업무" - 아침에 필요한 탭들 저장
2. "점심 쇼핑" - 쇼핑하면서 열어둔 탭들 저장
3. "저녁 공부" - 학습 자료 탭들 저장

필요할 때마다 바꿔가며 복구하면서 사용!

## ⚠️ 주의사항

### ❌ 할 수 없는 것

1. **저장된 세션 수정**
   - 세션 이름 변경 불가 (현재 버전)
   - 세션 내용 편집 불가
   - **해결책**: 필요한 세션은 복구 후 다시 저장

2. **개별 탭만 저장**
   - 모든 탭이 한 번에 저장됨
   - 일부 탭만 선택 저장 불가
   - **해결책**: 필요한 탭만 먼저 열고 저장

3. **삭제 취소**
   - 삭제한 세션 복구 불가
   - **주의**: 실수로 삭제하면 데이터 손실

### ✅ 안전한 것

1. **다른 탭에 영향 없음**
   - Stash해도 다른 브라우저 창의 탭은 영향 없음
   - 현재 창의 탭만 저장되고 닫힘

2. **로컬 저장**
   - 모든 데이터는 내 PC에만 저장
   - 클라우드 동기화 없음 (보안)
   - 다른 기기에서 접근 불가

3. **빠른 복구**
   - 몇 초 안에 탭 200개 이상도 복구 가능

## 🆘 문제 해결

### Q: 탭을 Restore했는데 안 보여요?
**A:** 
- Chrome 창의 탭 바를 확인하세요 (맨 위)
- 탭이 백그라운드에서 열려있을 수 있음
- 탭을 클릭하면 내용이 나타남

### Q: 세션을 실수로 삭제했어요
**A:**
- 안타깝게도 복구 불가 (현재 버전)
- 다음부터는 Preview로 확인 후 삭제하세요
- 미래 버전에서는 "실행 취소" 기능 추가 예정

### Q: 팝업창에 세션이 안 보여요?
**A:**
- 아래로 스크롤해서 확인하세요
- 팝업창이 작으면 세션이 짤릴 수 있음

### Q: PC가 재부팅되면 세션이 사라지나요?
**A:**
- 아니요! 모든 세션은 Chrome 저장소에 보관됨
- PC를 재부팅해도 세션은 유지됨
- Chrome을 완전히 지우지 않은 이상 안전

### Q: 몇 개까지 세션을 저장할 수 있나요?
**A:**
- 이론상 무제한 (Chrome 저장소 한도까지)
- 실제로는 500개 이상 저장 가능
- 과거 세션은 정기적으로 삭제해서 정리하세요

## 🔒 데이터 보안

### ✅ 어떻게 안전한가?

1. **로컬 저장**
   - 모든 데이터가 내 PC의 Chrome 저장소에만 저장
   - 인터넷 전송 없음 (100% 오프라인)
   - 다른 기기에서 접근 불가

2. **권한 최소화**
   - Chrome은 필요한 권한(탭, 저장소)만 요청
   - 개인정보 접근 권한 없음
   - 위치, 카메라, 마이크 권한 없음

3. **개인정보 수집 없음**
   - 사용자 추적 없음
   - 분석 데이터 수집 없음
   - Google에 데이터 전송 안 함

### ⚠️ 주의할 점

1. **같은 PC 사용자들이 접근 가능**
   - PC의 다른 사용자 계정에서도 Chrome의 세션 접근 가능
   - 공용 PC에서는 주의!

2. **Chrome 데이터 삭제 시 사라짐**
   - Chrome 검색 기록 삭제 시 함께 삭제될 수 있음
   - Chrome 완전 제거 시 모든 세션 삭제

3. **백업 없음**
   - 세션이 자동으로 백업되지 않음
   - 따라서 중요한 탭들은 북마크로도 저장하세요

## 📞 자주 묻는 질문

**Q: 확장 프로그램 설정은 어디에 있나요?**  
**A:** 현재 버전에서는 설정 옵션이 없습니다. (차후 추가 예정)

**Q: 여러 Chrome 윈도우에서 사용할 수 있나요?**  
**A:** 네! 한 Chrome 계정으로 여러 윈도우를 사용 중이면 모든 세션이 공유됩니다.

**Q: 탭의 스크롤 위치도 저장되나요?**  
**A:** 아니요. URL과 제목만 저장됩니다. 복구 후 스크롤은 최상단부터 시작합니다.

**Q: 로그인 상태도 유지되나요?**  
**A:** 일부 유지됩니다. 보안상 로그아웃되는 사이트도 있습니다.

**Q: 비슷한 기능이 Chrome에 있나요?**  
**A:** Chrome의 "탭 그룹" 기능과 비슷하지만, ToManyTab은 **탭을 닫으면서 저장**하는 것이 핵심입니다.
