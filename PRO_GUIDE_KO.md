## 🚀 Pro 업그레이드 방법

### 1단계: 아이콘 클릭
- 팝업창 우상단의 **⭐** 아이콘 클릭
- 또는 상단 우측의 **⚙️** 버튼 클릭

### 2단계: 업그레이드 모달 열기
![업그레이드 모달 열림](생략)

### 3단계: "Pro 업그레이드 👑" 버튼 클릭
- 팝업창이 나타나면 **"Pro 업그레이드"** 버튼 클릭

### 4단계: 결제 (실제 구현 시)
- 현재는 **테스트 모드**이므로 실제 결제 없음
- 30일 Pro 체험 자동 활성화 ✅

### 5단계: Pro 사용 시작! 🎉
- 이제 무제한 세션 저장 가능
- 광고 제거됨
- 우상단에 **👑 PRO (30일)** 배지 표시

## 💡 Pro 버전의 이점

### 1️⃣ 무제한 세션 저장
```
❌ Free (제한)
└─ 최대 5개 세션만 저장 가능
   6번째 세션을 저장하려면 기존 세션을 삭제해야 함

✅ Pro (무제한)
└─ 몇 개든 저장 가능
   세션 100개, 200개도 걱정 없음!
```

### 2️⃣ 깨끗한 인터페이스
- 광고 없음 → 더 깔끔한 화면
- 모든 UI가 깔끔하게 표시됨

### 3️⃣ 클라우드 동기화 (예정)
- 다른 기기에서도 같은 세션 사용 가능
- PC, 노트북, 회사 컴퓨터 모두 동기화
- 세션이 자동으로 백업됨

### 4️⃣ 우선 지원
- 문제가 생기면 빠른 응답
- 새 기능 먼저 이용 가능

## 🧪 현재 상태 (테스트 모드)

### 현재 구현된 것
✅ Free vs Pro 구분  
✅ 세션 개수 제한 (Free: 5개, Pro: 무제한)  
✅ 업그레이드 모달  
✅ Pro 상태 표시 (배지)

2. **무제한 세션 확인**

3. **Pro 배지 확인**
   - 우상단에 **👑 PRO (29일)** 같은 표시
   - 일부러 시스템 시간을 조정하면 만료 테스트 가능

## 📱 실제 결제 시 필요한 것

현재는 **테스트 코드**이므로 실제 결제 미포함.  
실제 배포 시 다음이 추가됩니다:

### Stripe 결제 연동 (권장)
```
1. Stripe 계정 생성
2. API Key 설정
3. 결제 폼 연동
4. 영수증 관리
5. 구독 관리 (자동 갱신, 취소 등)
```

### 결제 플로우
```
"Pro 업그레이드" 버튼 클릭
    ↓
Stripe 결제 페이지 열림
    ↓
카드 정보 입력
    ↓
$1.99 결제
    ↓
Pro 활성화 (30일)
    ↓
자동 갱신 (매월 $1.99)
```

## 🔄 Pro 관리

### Pro 확인하기
현재 Pro 상태 확인:
1. 팝업창 우상단 확인
   - ⭐ = Free 사용자
   - 👑 PRO = Pro 사용자

### Pro 취소하기 (예정)
실제 구현 시 추가될 기능:
1. 설정에서 "Pro 관리" 선택
2. "Pro 취소" 버튼 클릭
3. 구독 취소 완료
4. 현재 기한까지는 Pro 유지
5. 이후 Free로 자동 전환

### 만료되면?
Pro 기간 만료 시:
- 자동으로 Free로 전환
- 저장된 세션은 유지
- 6개 이상 세션은 사용 제한
- (가장 최근 5개만 접근 가능)

## ⚠️ 주의사항

### ❓ Pro가 필요한가요?

**Pro가 필요한 경우:**
- 🔴 현재 5개 이상 세션을 자주 써야 함
- 🔴 시간이 지나면서 세션이 계속 쌓임
- 🔴 다양한 프로젝트를 동시에 진행 중
- 🔴 장기간 세션을 보관해야 함

**Free로도 충분한 경우:**
- 🟢 주로 5개 이하 세션만 사용
- 🟢 오래된 세션은 주기적으로 삭제
- 🟢 일시적으로 탭만 정리하고 싶음

### 💰 구독 해지 가능

언제든지 Pro 구독을 취소할 수 있습니다:
- 환불 정책 적용
- 취소 후 기한까지 Pro 유지
- 이후 Free로 자동 전환
- **강압적 구독 없음** ✅

## 📝 FAQ

**Q: 현재 Pro로 업그레이드하면 결제되나요?**  
**A:** 아니요! 현재는 테스트 모드입니다. 30일 Pro를 무료로 체험할 수 있습니다.

**Q: Pro를 취소하면 세션이 삭제되나요?**  
**A:** 아니요! 세션은 유지됩니다. 다만 6개 이상 세션은 접근 제한됩니다.

**Q: 여러 기기에서 Pro 사용 가능한가요?**  
**A:** 현재는 설명서에 없지만, 클라우드 동기화가 추가되면 가능합니다.

**Q: 환불 정책이 뭔가요?**  
**A:** 구독 후 30일 이내 취소 시 100% 환불 가능 (예정).

**Q: Pro는 수동 갱신인가요?**  
**A:** 아니요, 자동 갱신입니다. 취소하기 전까지 매월 청구됩니다.
